Pandas Basics - Getting up-and-running

Jun 7, 2020 • 19 min read

This notebook will get you to a point where you can quickly explore a dataset along with basic dataset editing.

You can simply load Pandas using "Import Pandas", however to save time when calling the module we load it with the alias "pd"

# Import Pandas
import pandas as pd

For this example we will use an IMDB dataset that can downloaded from Kaggle here

The dataset is formatted as CSV so we need to call the .read_csv() function. If it was saved as an Excel file we could use the .read_excel() function.

# Import IMDB dataset
df = pd.read_csv('IMDb movies.csv')
